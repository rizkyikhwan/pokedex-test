## How do run this repository

1. Clone or Download this repository
2. Open project

### `yarn install`

3. Type "yarn install" on the terminal

### `yarn dev`

4. Type "yarn dev" for running the project

### Dependencies

Node Version 18.18.1

1. Vite
2. React + Typescript
3. Tailwind CSS
4. Zustand
5. React-icons
6. React Router DOM v6
7. Axios
8. clsx
